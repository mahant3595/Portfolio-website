# # Personal Portfolio Website

## Description
This is a full-stack portfolio website to showcase projects.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MySQL

## How to Run

### 1. Setup Database
Run schema.sql in MySQL

### 2. Backend
cd backend
npm install
node server.js

### 3. Frontend
Open index.html in browser
